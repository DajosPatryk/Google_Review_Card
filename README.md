# Google_Review_Card

A pretty Google Review Card made with Vuetify and vue2-google-maps library.
Can be used on any vue app you desire, no licensing or credits required.
Example: https://prod.d1oc8xgks2d2bl.amplifyapp.com/

Make sure to install the necessary dependencies:
#1.1 vue add vuetify
OR 
#1.2 npm i vuetify
#2 npm i vue2-google-maps
#3 Configure vue2-google-maps by adding it to your project and adding your license key.
